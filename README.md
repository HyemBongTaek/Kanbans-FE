## 프로젝트 소개

코코리는 coworker collaboration remember의 줄임말로
동료와 함께 협업하고 기록하자는 의미에서 코코리라고 이름지었습니다. 
코코리는 시간관리를 위한 뽀모도로 타이머와 
프로젝트에 맴버를 추가해 함께 사용 가능한 보드를 제공합니다.
